🔊Permanent Audio MP3

Bahagian ini disediakan supaya cikgu boleh masukkan audio MP3 sendiri secara kekal. Letakkan fail MP3 dalam folder audio, kemudian masukkan nama fail dalam bahagian permanentAudio di dalam kod JavaScript.

1. Simpan fail MP3 Contoh: audio/ba.mp3, audio/e-pepet.mp3, audio/be.mp3
2. Masukkan dalam kod Isi nama audio di bahagian const permanentAudio = { ... }
3. Klik kad Kad akan mainkan MP3 dahulu. Jika MP3 tiada, suara browser akan digunakan semula.
const permanentAudio = { a: "audio/a.mp3", "ǝ": "audio/e-pepet.mp3", e: "audio/e-taling.mp3", b: "audio/b.mp3", ba: "audio/ba.mp3", "bǝ": "audio/b-pepet.mp3", be: "audio/be.mp3" };
✅ Fungsi asal dikekalkan + MP3 sendiri ditambah.
